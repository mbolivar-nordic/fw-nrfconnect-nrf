Configuration Reference
#######################

Application configuration in the nRF Connect SDK uses two domain-specific
languages: **Kconfig** and **devicetree**. This documentation contains
reference material for available Kconfig options and devicetree bindings. Its
contents are generated automatically from the Kconfig and devicetree files
themselves.
